✅ This will now work automatically! The app detects all <br>, <br/>, and <br /> tags within node labels and normalizes them so Mermaid renders them as actual line breaks (not visible text).

How It Works

The improved normalizeBrTags() function:

  • Uses a stack-based parser to track when code is inside node labels
  • Automatically detects labels within [], (), {}, and ""
  • Normalizes all <br> variants to <br/> within labels
  • Mermaid then renders these as actual line breaks in the diagram
  • Preserves all other syntax unchanged
  • Works across multiple lines and with nested structures

Supported Break Tag Formats

Result: All variations are normalized and rendered as proper line breaks in the final diagram. You'll see multi-line text in the nodes, not the literal "<br>" text.
